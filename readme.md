# Sistema de colas V2


## Generar un ticket

Para ver la pantalla "generar tickets" (botones para crear un ticket a X área)
URL:  / o /colasv2


Se necesita de una cuenta o estar logeado. Este usuario tiene el rol Ticket y sólo puede ingresar a esta vista.

username: ticket1234
password: ticket1234


## Atender un ticket

Es el panel donde se atienden los tickets. El usuario logeado tiene el rol Trabajador y
pertenece a una área.

URL: / o /colasv2/manejo-de-tickets

username: tesorero1234
password: 1234567890


## Monitor

Pantalla para ver el turno. Se necesita de una cuenta o estar logeado. Este usuario tiene el rol Monitor y sólo puede ingresar a esta vista.

URL: / o /colasv2/monitor

username: monitor1234
password: 1234567890


### Conclusiones

Con sólo ingresar a la raíz /, colocar el usuario y contraseña correctos, se puede ingresar a las vistas.
- Con el usuario de monitor, se puede ver el turno.
- Con el usuario de trabajador, se puede atender tickets.
- Con el usuario de ticket, se puede generar tickets.
